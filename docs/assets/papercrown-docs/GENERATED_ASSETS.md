# Generated Docs Art Assets

This folder contains generated art for the Paper Crown documentation anthology.

- `covers/`: generated front cover art.
- `dividers/`: one shallow panoramic chapter header plate for each anthology
  genre.
- `splashes/`: in-chapter showcase splash art for selected reference sections.
- `ornaments/`: illustrated headpieces and tailpieces.
- `fillers/`: auto-placeable conditional filler art for PDF builds.
- `page-wear/`: transparent scratch, smudge, and smear overlays using Paper
  Crown naming conventions.

The generated images avoid embedded readable text so headings, captions, and accessibility remain controlled by Markdown and CSS.
